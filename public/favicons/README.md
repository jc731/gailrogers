# Favicon Files

This directory contains favicon files for the website.

## Files

- `favicon.svg` - Modern SVG favicon (scalable, preferred by modern browsers)
- `favicon.ico` - Traditional ICO format (for older browsers)
- `apple-touch-icon.png` - Apple touch icon (180x180px for iOS devices)
- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `android-chrome-192x192.png` - Android Chrome icon (192x192px)
- `android-chrome-512x512.png` - Android Chrome icon (512x512px)

## Generation

The SVG favicon is the source. Other formats can be generated from it using tools like:
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)
- Image editing software

## Current Icon

The favicon uses a simplified version of the logo's classical courthouse building icon, maintaining brand consistency.
